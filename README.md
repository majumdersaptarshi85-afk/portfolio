# Premium React Portfolio

A modern, interactive React portfolio website with cursor-based animation and emotional photo reactions.

## Setup

1. Install dependencies
   ```bash
   npm install
   ```
2. Run the development server
   ```bash
   npm run dev
   ```

## Notes
- Replace `public/profile.svg` with your own photo or update the `InteractiveProfilePhoto` component to load a real image.
- The site uses Tailwind CSS and Framer Motion for smooth animations.
